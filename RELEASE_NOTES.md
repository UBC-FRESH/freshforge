# Release Notes

## 0.1.0a1

Initial public bootstrap target for FreshForge.

Planned contents:

- package-backed Python scaffold;
- minimal CLI;
- strict FRESH roadmap, issue, and changelog workflow;
- Sphinx docs skeleton;
- CI, docs, and release-artifact workflow scaffolding.

FreshForge `0.1.0a1` does not implement a workflow DSL or execution engine.
