# Code Of Conduct

FreshForge follows the UBC FRESH Lab expectation that project spaces remain
professional, respectful, and focused on constructive scientific and engineering
work.

Contributors should communicate respectfully, assume good faith while being
precise about technical issues, keep public issues and pull requests free of
private material, and report problems to the project maintainers.

This lightweight code of conduct may be replaced by a fuller adopted standard in
a later governance phase.
